Welcome to the Better Than Solo server!

This is a friendly (non-anarchy) server set up shortly after Better Than Wolves went SMP, because it is way more fun to play BTW together! :)

I'm actively modding the server in order to monitor performance and take action when something is slowing the server down. I've also made some changes which you can read about below.

Guide for New Players:

Since this was one of the first SMP servers, there is a fair bit of craziness around spawn. There are lots of old bases and abandoned construction projects from when the server had 20-30 people online at once.

First, I recommend taking a look around the spawn town. You are free to take any supplies in chests that you can open (some are locked for players who build at spawn). However, since they're intended to be starter supplies for new players, you shouldn't just empty them out. Take what you need in the short term and leave the rest for players to come!

Second, take a look at the server's Overviewer Map. Using the day/night toggle in the upper right, try to find a place on the map where there doesn't seem to be any lights on the surface. Then adventure to that area and start your base :)

If you're not sure what to do or you need help, feel free to post to the forum thread!
